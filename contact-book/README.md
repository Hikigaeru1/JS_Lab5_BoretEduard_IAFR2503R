# Индивидуальный проект: Книга контактов

## Описание проекта

**Книга контактов** — это веб-приложение, разработанное на чистом JavaScript (Vanilla JS). Проект представляет собой простой телефонный справочник, в котором пользователь может просматривать список контактов, добавлять новые записи, удалять контакты, выполнять поиск, сортировку и фильтрацию по группам.

Цель проекта — закрепить практические навыки работы с JavaScript, DOM-деревом, событиями, массивами объектов, пользовательским вводом и модульной структурой кода без использования сторонних фреймворков.

## Основные функции

В проекте реализованы следующие возможности:

- отображение начального списка контактов;
- добавление нового контакта через модальное окно;
- удаление контакта из списка;
- поиск контактов по имени, телефону или email;
- сортировка контактов по имени, телефону или email;
- фильтрация контактов по группам: «Семья», «Друзья», «Работа»;
- валидация пользовательского ввода;
- вывод сообщений об ошибках при неправильном заполнении формы;
- динамическое создание HTML-элементов через JavaScript;
- работа с DOM-событиями;
- адаптивный интерфейс для разных размеров экрана.

## Структура проекта

```text
contact-book/
│
├── index.html
├── style.css
│
└── js/
    ├── main.js
    ├── data.js
    ├── contacts.js
    ├── dom.js
    └── validation.js
```

### Назначение файлов

- `index.html` — основная HTML-страница приложения.
- `style.css` — стилизация интерфейса.
- `js/main.js` — главный файл приложения, связывает все модули и обработчики событий.
- `js/data.js` — начальный массив контактов.
- `js/contacts.js` — функции для добавления, удаления, поиска, сортировки и фильтрации контактов.
- `js/dom.js` — функции для работы с DOM: открытие/закрытие модального окна, очистка формы, создание карточек контактов и отрисовка списка.
- `js/validation.js` — проверка данных формы перед добавлением нового контакта.

## Установка и запуск проекта

### Необходимые зависимости

Для запуска проекта не требуются сторонние библиотеки или фреймворки. Достаточно современного браузера и, желательно, редактора Visual Studio Code.

Используемые технологии:

- HTML5;
- CSS3;
- JavaScript ES6+;
- JavaScript Modules.

### Запуск через Live Server

Так как проект использует модульное подключение JavaScript через `type="module"`, рекомендуется запускать его через локальный сервер.

1. Скачайте или клонируйте проект.
2. Откройте папку проекта в Visual Studio Code.
3. Установите расширение **Live Server**, если оно ещё не установлено.
4. Откройте файл `index.html`.
5. Нажмите правой кнопкой мыши по файлу и выберите **Open with Live Server**.
6. Приложение откроется в браузере.

Пример адреса при запуске:

```text
http://127.0.0.1:5500/index.html
```

### Альтернативный запуск

Если модульные импорты корректно поддерживаются окружением, проект можно открыть через локальный сервер любым другим способом, например через расширение Live Server или встроенный сервер редактора.

## Примеры использования

### Добавление контакта

1. Нажмите кнопку **«Добавить контакт»**.
2. В открывшемся модальном окне заполните поля:
   - имя;
   - телефон;
   - email;
   - группа.
3. Нажмите кнопку **«Сохранить»**.
4. Новый контакт появится в общем списке.

Пример объекта нового контакта:

```js
const newContact = {
    id: Date.now(),
    name: name.trim(),
    phone: phone.trim(),
    email: email.trim(),
    group: group
};
```

### Поиск контакта

Для поиска контакта введите текст в поле поиска. Поиск выполняется по имени, номеру телефона и email.

Пример логики поиска:

```js
export const searchContacts = (contacts, searchText) => {
    const text = searchText.toLowerCase();

    return contacts.filter(contact => {
        return (
            contact.name.toLowerCase().includes(text) ||
            contact.phone.toLowerCase().includes(text) ||
            contact.email.toLowerCase().includes(text)
        );
    });
};
```

### Фильтрация контактов

Контакты можно отфильтровать по группе:

- все группы;
- семья;
- друзья;
- работа.

Пример функции фильтрации:

```js
export const filterContactsByGroup = (contacts, group) => {
    if (group === "all") {
        return contacts;
    }

    return contacts.filter(contact => contact.group === group);
};
```

### Сортировка контактов

В приложении доступна сортировка по:

- имени;
- телефону;
- email.

Пример функции сортировки:

```js
export const sortContacts = (contacts, sortType) => {
    const sortedContacts = [...contacts];

    sortedContacts.sort((a, b) => {
        if (a[sortType] > b[sortType]) {
            return 1;
        }

        if (a[sortType] < b[sortType]) {
            return -1;
        }

        return 0;
    });

    return sortedContacts;
};
```

## Валидация формы

Перед добавлением контакта выполняется проверка введённых данных. Программа проверяет:

- заполнено ли имя;
- заполнен ли телефон;
- содержит ли телефон минимум 6 символов;
- заполнен ли email;
- содержит ли email символ `@`;
- выбрана ли группа контакта.

Если данные введены неправильно, контакт не добавляется, а пользователь видит сообщение об ошибке.

## Особенности реализации

В проекте используется модульный подход. Код разделён на несколько файлов, каждый из которых отвечает за отдельную часть логики приложения. Это упрощает чтение, поддержку и расширение проекта.

Проект реализован без сторонних библиотек и фреймворков, что позволяет лучше понять базовые механизмы JavaScript: работу с DOM, событиями, массивами объектов и пользовательским вводом.

Также в приложении используется модальное окно для добавления контактов. Оно открывается по нажатию на кнопку и закрывается при нажатии на крестик или на затемнённую область вокруг формы.
